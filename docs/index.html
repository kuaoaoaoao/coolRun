coolRun

轻量的 macOS 菜单栏系统监控与金价工具。实时查看 CPU、内存、储存、电池、网络状态，以及浙商银行积存金价格。

MENUBAR 菜单栏常驻 不占 Dock，不打断当前工作。

SYSTEM 系统监控 CPU、内存、储存、电池、网络。
OPEN MIT 开源 源码公开，可自由学习和修改。

把 Mac 的状态压缩成一个小入口

coolRun 的重点不是做一个复杂仪表盘，而是把每天高频查看的信息放到最顺手的位置。

CPU 金币动态指示

CPU 占用越高，菜单栏金币旋转越快，不打开面板也能感知负载。

RAM 系统监控面板

集中展示 CPU、内存、储存、电池和网络状态，信息密度高但不拥挤。

¥/g 浙商积存金价格

菜单栏直接显示金价，不需要用户配置 API Key。

UI 浅色与深色模式

使用 macOS 系统自适应颜色，跟随系统外观保持清晰。

安装与使用

从 GitHub Releases 下载应用，拖入 Applications 文件夹后启动即可。

下载安装包

进入 Releases 页面，下载最新的 coolRun.dmg 或 coolRun.zip。

安装应用

打开 DMG，将 coolRun.app 拖入 Applications 文件夹。

开始使用

启动后在菜单栏找到金币图标，左键查看监控，右键打开快捷菜单。

如果 macOS 提示无法验证开发者，可以右键点击 coolRun.app，选择“打开”，再在系统弹窗中确认。
